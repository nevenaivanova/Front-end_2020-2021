HTML

HTML(Hypertext Markup Language) is the set ot markup symbols ott codes inserted in a file intended for display on a World Wide Web browser page. The markup tells Web browser how to dispay a Web page's words and images for the user. Each individual markup code is referred to as an element (nut many people also refer to it as a tag). Some elements come in pairs that indicate when some display effect is to begin and whe it is to end.

CSS

CSS is a stylesheet langiage that describes the presentation of an HTML(or XML) documents. CSS describes how elements must be rendered on screen, om paper, or in other media.
